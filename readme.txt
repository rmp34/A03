
Step by Step Tutorial:
1) Go to https://www.jetbrains.com/student/ to download webstorm.
2) Go to https://git-scm.com/downloads to download git.
3) Go to https://github.com/join to create a github account.
4) Connect github with webstorm. In webstorm, press (Ctrl+Alt+S) for system preferences. Select Version control Git. Enter the path to the git.exe.
5) Add github password to webstorm. In Webstorm press (Ctrl+Alt+S) for system preferences.Select Appearance and Behavior, System Settings, then Passwords.
6) Create a repository in github. Click the + sign in the upper right corner. Choose “Create New repository”.
7) Make the repository public and add the readme file. Click Create.
8) Create a repository from webstorm. Select VCS and Import into Version Control.
9) Import a repository from github. From Main page Select Checkout from version control, Git  OR
	From within Webstorm Select VCS, Checkout from version control, Git. Enter Github repository name, then Enter local path name.
10) Create a webstorm file. Choose File, then HTML, HTML 5 or , File, then Stylesheet.
11) add files to git. The Add to Git dialog opens.Click Add. This adds to local file system.
12) commit your changes.
13) Push Change to Remote Repository. Click “Ctrl Shift  K”Or “VCS Git  Push”
14) File is on github.
15) set up github pages. Click Settings, then Check the repository name.
16) Choose GitHub Page Location. Select “Master branch”, Note the published URL 
17) Check your GitHub Pages. Copy the Github.io URL into a browser, then Post the URL into canvas with your Github account URL
