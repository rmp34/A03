
Step by Step Tutorial:
1) Go to https://www.jetbrains.com/student/ to download webstorm.
2) Go to https://git-scm.com/downloads to download git.
3) Go to https://github.com/join to create a github account.
4) Connect github with webstorm. In webstorm, press (Ctrl+Alt+S) for system preferences. Select Version control Git. Enter the path to the git.exe.
5) Add github password to webstorm. In Webstorm press (Ctrl+Alt+S) for system preferences.Select Appearance and Behavior, System Settings, then Passwords.
6) Create a repository in github. Click the + sign in the upper right corner. Choose “Create New repository”.
